# 1 Purpose

This document defines the Information Architecture (IA) of the Japan Sourcing Platform (JSP).

It specifies how information is organized, navigated, and presented to users.

This document serves as the foundation for:

- Website Navigation
- UI Design
- JSP Page Structure
- URL Design
- SEO Structure

---

# 2 Design Principles

The information architecture follows these principles:

Simple

Professional

Easy Navigation

Mobile Friendly

SEO Friendly

Catalog First

Inquiry Oriented
